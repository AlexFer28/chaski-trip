# Chaski Trip 🚗

Chaski Trip 🚗 es una aplicación web para planificar viajes en el Peru.

## 🚀 Inicio

### 📦 Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/AlexFer28/chaski-trip.git
```

2. Instalar dependencias

```bash
bun install
```

3. Iniciar el servidor de desarrollo

```bash
bun dev
```

4. Abrir la aplicación en el navegador

```bash
open http://localhost:5173
```
